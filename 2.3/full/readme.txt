Установка модуля через OCMOD:
 
1. Для установки модуля в настройках должен быть включён FTP.
2. Если при включении FTP появляется ошибка - возможно поможет решение: https://www.opencart.com/index.php?route=marketplace/extension/info&extension_id=18892
3. Открыть из админки "Модули->установка расширений->загрузить файл(apimobile.ocmod.zip).
